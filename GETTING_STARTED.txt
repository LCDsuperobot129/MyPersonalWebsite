🎯 3-STEP CUSTOMIZATION
═══════════════════════════════════════════════════════════════════════════════

  1. Edit these 3 data files:
     → src/data/projects.ts        (Your projects)
     → src/data/experience.ts      (Your experience)
     → src/data/skills.ts          (Your skills)

  2. Update personal info:
     → src/components/HeroSection.tsx  (Name, links)
     → src/app/layout.tsx              (SEO metadata)

  3. Add your resume:
     → public/resume.pdf           (Your resume PDF)

📦 PROJECT STRUCTURE
═══════════════════════════════════════════════════════════════════════════════

  src/
  ├── app/                    → Next.js App Router
  │   ├── layout.tsx          → Root layout + metadata

  ├── components/             → React components

  │   ├── projects.ts
  │   ├── experience.ts
  │   └── skills.ts
  ├── types/                  → TypeScript types
  └── lib/                    → Utilities

  public/
  └── resume.pdf              → ADD YOUR RESUME HERE
